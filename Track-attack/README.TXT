Track Attack
------------

Camera views :

F1 : Cockpit view
F2 : Behind car view (Toggles between near, mid and far)
F3 : Overhead view (6, 7, 8, 9 Zooms)
F4 : Bumper view
F5 : Rotating view (Home, End, Del, Page Down rotates)
                   (6, 7, 8, 9 Zooms)

Other options :

F8 : Damage on/off
F12 : Toggles perspective texturing

+/- : Changes screen size

N.B. Game starts in pause mode.
